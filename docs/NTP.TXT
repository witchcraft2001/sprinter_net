NTP.EXE - network time setup

Purpose
  Gets the current network time and sets the DSS system clock.

Usage
  NTP.EXE

Configuration
  NTP reads these values from NET.CFG:

  TZ   Time zone offset, for example +6, +5:30 or -3:30.
  NTP  Time server name, for example pool.ntp.org.

Typical sequence
  NETUP.EXE
  NTP.EXE

Exit status
  0  Time was received and DSS SETTIME succeeded.
  2  Sprinter-WiFi hardware was not found.
  3  ESP communication, SNTP, parse or DSS SETTIME error.

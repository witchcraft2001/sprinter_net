TCPTEST.EXE - TCP smoke test

Purpose
  Opens a TCP connection and prints a short HTTP response. Use it to check the
  connection after NETUP.

Notes
  TCPTEST is a smoke test, not a downloader. Console output is intentionally
  limited to a few receive blocks, so long HTML pages may be truncated.

Usage
  TCPTEST.EXE [host [port [path]]]

Defaults
  host: example.com
  port: 80
  path: /

Examples
  TCPTEST.EXE
  TCPTEST.EXE example.com 80 /
  TCPTEST.EXE 192.168.1.36 80 /check-2k.txt

Typical sequence
  NETUP.EXE
  TCPTEST.EXE

Role
  TCPTEST is a diagnostic utility, not a downloader. It is useful before using
  WGET, FTP or TFTP.

Exit status
  0  Test completed.
  2  Sprinter-WiFi hardware was not found.
  3  ESP/TCP communication error.

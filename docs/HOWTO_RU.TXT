Sprinter ESP Network Kit - Быстрый старт (HOWTO)
==================================================

Что нужно
  - Sprinter DSS с картой SprinterESP / Sprinter-Wi-Fi (ESP8266, ESP-AT).
  - Прошивка ESP-AT версии 2.2.1. Это единственная проверенная версия,
    на других работа не гарантируется. Проверить версию можно программой
    NETPROBE.
  - Имя и пароль вашей сети Wi-Fi.

1. Установка
  Распакуйте пакет в C:\WIFI (или используйте уже установленную копию).
  Добавьте каталог пакета в PATH в файле C:\SYSTEM.BAT, чтобы все утилиты
  можно было запускать из любого каталога. Добавьте строку, не удаляя уже
  существующие каталоги из PATH:

      SET PATH=%PATH%;C:\WIFI

  Перезагрузите DSS или один раз запустите C:\SYSTEM.BAT, чтобы применить
  изменение. Затем перейдите в каталог пакета для первоначальной настройки:

      C:
      CD \WIFI

2. Настройка Wi-Fi  (делается один раз)
  Запустите интерактивный редактор и впишите параметры своей сети:

      NETCFG /W

  Задайте:
      SSID = имя вашей сети Wi-Fi
      PASS = пароль вашей сети Wi-Fi
      DHCP = 1            (для обычного домашнего роутера оставьте 1)
      BAUD = 115200       (поставьте 57600 или 38400, если на 115200 теряются
                           байты)
  Сохраните. Настройки хранятся в C:\WIFI\NET.CFG.

3. Поднимите соединение
      NETUP

  При успехе выводятся профиль прошивки ESP, назначенный IP (например,
  192.168.1.138) и "NETUP done", а также публикуются переменные окружения
  NET_* (NET=WIFI, NET_ESP_FW=2.2.1 или 2.2.2, NET_IP, NET_GW, ...) для
  других программ.

4. Проверьте, что всё работает
      PING 8.8.8.8
      WGET http://example.com -o INDEX.HTM -y

  Успешный PING печатает время ответа; WGET сохраняет страницу в INDEX.HTM.

Запуск одной командой (batch)
  CONNECT.BAT последовательно выполняет NETRESET, NETUP и PING 8.8.8.8.

Повседневные команды (запускать после NETUP)
  PING host                 проверка доступности
  WGET url [-o file]        скачать файл по http:// (HTTPS не поддерживается)
  FTP  host file [-u user -p pass]   скачать по FTP
  TFTP host GET file -o FILE          скачать по TFTP
  NTP                       установить часы DSS по сети
                            Сервер NTP и таймзона берутся из NET.CFG
                            (NTP=, TZ=).
  WTERM                     открывает ручной терминал ESP-AT для ввода
                            команд вручную.

  Перезапись/докачка существующего локального файла (WGET, FTP, TFTP):
    если файл существует, утилита спрашивает [R]esume / [O]verwrite / [C]ancel.
    -y или -f   перезаписать без вопроса
    -r          докачать/дописать без вопроса (WGET, FTP; в TFTP докачки нет)
  Удобно в batch-файлах, например:  WGET url -o FILE.BIN -y

Если что-то зависло и программы выдают ошибки подключения "ESP connection
error #1"
  Выполните по порядку:
      NETRESET      сброс модуля ESP
      NETUP         переподключение

Советы
  - Запускайте NETUP один раз за сеанс; остальные утилиты используют это же
    соединение.
  - Если утилита сразу после NETUP сообщает "busy", сетевой стек ESP ещё
    поднимается - просто запустите её ещё раз.
  - Поддерживается только http://, не https://.
  - Используйте CONNECT.BAT для подключения, этот файл содержит нужные команды
    для инициализации и подключения сети после запуска компьютера.

Подробнее
  READMERU.TXT - обзор, полный список команд, примеры batch-файлов.
  USAGE.MD     - полный справочник, ключи конфигурации, рабочий процесс, коды
                 завершения.
  <TOOL>.TXT   - по одной странице на утилиту (NETUP.TXT, WGET.TXT, FTP.TXT,
                 ...).

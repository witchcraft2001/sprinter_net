================================================================
  Sprinter ESP Network Kit
  Сетевой пакет для Sprinter DSS и Wi-Fi-карты SprinterESP
  (ESP12-F / ESP8266, прошивка ESP-AT).      Версия 0.1.1
================================================================

Набор утилит для выхода Sprinter в сеть через Wi-Fi-карту
SprinterESP: настройка и подъём соединения, диагностика, загрузка
файлов (HTTP / FTP / TFTP), синхронизация времени и терминал ESP-AT.

ВАЖНО: пакет рассчитан на прошивку ESP-AT версии 2.2.1 — только она
проверена, на других версиях работа не гарантируется. Проверить
версию прошивки: запустите NETPROBE (строка AT+GMR) или в WTERM
введите  AT+GMR.


С ЧЕГО НАЧАТЬ
------------
  1. Распакуйте пакет в C:\WIFI и перейдите в каталог:
         C:
         CD \WIFI
  2. Добавьте каталог пакета в PATH в файле C:\SYSTEM.BAT, чтобы утилиты
     запускались из любого каталога. Добавьте строку, сохранив старый PATH:
         SET PATH=%PATH%;C:\WIFI
     Перезагрузите DSS или один раз запустите C:\SYSTEM.BAT для применения.
  3. Настройте Wi-Fi (один раз):   NETCFG /W
         SSID и пароль вашей сети, DHCP=1, BAUD=115200
  4. Поднимите соединение:         NETUP
  5. Проверьте связь:              PING 8.8.8.8

  Подробное руководство ...... файл DOCS\HOWTO_RU.TXT
  Справка по утилите ......... файл DOCS\<ИМЯ>.TXT  или  <ИМЯ> /?
  Полный справочник .......... файл DOCS\USAGE.MD


ПРОГРАММЫ
---------
  NETCFG.EXE    редактор сетевой конфигурации (файл NET.CFG)
  NETUP.EXE     инициализация карты и подъём соединения
  NETRESET.EXE  сброс и переинициализация ESP
  NETPROBE.EXE  низкоуровневая диагностика ESP
  PING.EXE      проверка доступности узла
  WGET.EXE      загрузка файлов по HTTP
  FTP.EXE       передача файлов и листинг по FTP (пассивный режим)
  TFTP.EXE      клиент TFTP
  NTP.EXE       синхронизация системного времени по сети
  WTERM.EXE     терминал ESP-AT (ручной ввод AT-команд)
  TELNET.EXE    ANSI Telnet-клиент с передачей файлов Zmodem


ДОКУМЕНТАЦИЯ  (каталог DOCS\)
-----------------------------
  HOWTO_RU.TXT  быстрый старт (русский)
  HOWTO.TXT     быстрый старт (английский)
  USAGE.MD      полный справочник и коды возврата
  <ИМЯ>.TXT     справка по каждой утилите: PING.TXT, WGET.TXT,
                FTP.TXT, TFTP.TXT, NTP.TXT, NETCFG.TXT, NETUP.TXT,
                NETRESET.TXT, NETPROBE.TXT, WTERM.TXT, TELNET.TXT


КОНФИГУРАЦИЯ  (каталог CONFIG\)
-------------------------------
  NETSMPL.CFG   образец NET.CFG. Скопируйте в NET.CFG или запустите
                NETCFG /W.


ПРИМЕРЫ  (каталог EXAMPLES\)
----------------------------
  CONNECT.BAT   инициализация карты, подъём соединения и проверка связи
  TFTPGET.BAT   приём файла по TFTP
  TFTPPUT.BAT   передача файла по TFTP
